# VideoGamesShop
---
Данный проект является прототипом онлайн магазина, нацеленного на продажу лицензионных версий компьютерных игр.
---
Функционал проекта:
* Просмотр каталога видеоигр;
* Оформление покупки желаемого товара;
* Просмотр "библиотеки" приобритенных товаров;
* Возврат средств за покупку.
---
Аналоги:
* Steam
* Epic Games Store
* GOG
